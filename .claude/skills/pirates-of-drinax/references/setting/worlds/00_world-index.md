## Legend
Columns: **World | Hex | UWP | Bases | Trade Codes | Zone | Allegiance | Notable**.
- **Hex** — 4-digit column/row on the subsector map (e.g. `0603` = col 06, row 03). Travel distance = parsecs between hexes.
- **UWP** — `Starport-Size-Atmo-Hydro-Pop-Gov-Law–Tech`. Decode each digit in **rules/world-and-system-gen.md**: Starport (A excellent shipyard/refined fuel … E frontier no fuel, X none); Atmo (0 vacuum, 6 standard, A exotic, B corrosive, C insidious); Hydro (A = waterworld); Pop (zeroes after a 1; 0 = uninhabited); Gov; Law (what's contraband + how often you're stopped); Tech (after the dash; 12+ = High Tech, F=15). Base-16 digits: A=10…F=15, occasional G=16.
- **Bases** — N Naval, S Scout, W Way-Station, C Corsair, D Depot, M Military, R Research; **ACS/AIC/ACF/ARS/IRS** are Aslan-port shorthand from the gazetteer (A=Aslan port + Clan/Imperial + Scout/Corsair/etc.); treat any unlisted letter as a minor installation (see decoder "Bases").
- **Trade Codes** — Ag As Ba De Fl Ga Hi Ht Ic In Lo Lt Na Ni Po Ri Va Wa; each implies buy/sell DMs and conditions — full table in the decoder.
- **Zone** — blank/Green = normal; **A** = Amber (flagged dangerous); **R** = Red (interdicted, entry is itself a crime).
- **Allegiance** — controlling power (Imperium, Aslan Hierate, Glorious Empire, Florian League, Kingdom of Drinax, etc.); blank = independent/unaligned.
- **Notable** — one phrase: campaign role, hazard, attitude to Drinax, or fighting strength. For New-Kingdom candidate worlds the **Attitude** (Hostile→Haven) and **FS** (Fighting Strength 0–5+) come from the *Appendixes* recruitment table; use them for the empire game (FS: 0 = no ships, 2 = average backwater, 4 = Theev-tier stronghold/real navy, 5+ = massive military).
- `(verify)` marks a cell unreadable in every source; a digit in **bold** inside Notable flags a cross-source conflict resolved in favour of the gazetteer (variants listed under "Conflict notes").

## Cross-source conflict notes
Where the **Appendixes** recruitment table and the **gazetteer** (PoD2) disagree, this index uses the **gazetteer UWP** as canonical (it matches the project's own `setting/trojan-reach.md`), and lists the Appendix variant in the world's Notable cell. Material differences worth knowing in play:
- **Drinax** 0603: gaz `A43645A-E` vs Appx `A33645C-F` (Appx string is malformed — extra digit before the dash; gaz used).
- **Tyokh** 0606: gaz `B566A77-E` vs Appx `B466AH7-E` (Appx has illegible `H`; gaz used).
- **Khusai** 0403: gaz `A576655-C` vs Appx `A5766M5-C` (illegible `M`); **Kteiroa** 0505: gaz `C560164-8` vs Appx `C4601K4-8` (illegible `K`); **Iroioah** 0607: gaz `B530113-C` vs Appx `B6301G3-E` (illegible `G`) — gaz used in all three.
- **Ace** 0109: gaz `E7A0889-9` vs Appx `E7A08B9-5` (Appx malformed) — gaz used.
- **Oghma** (Sindal) 0410: gaz `B534754-9` vs Appx `B214754-9` — size/atmo/hydro differ; the `Ic` trade code fits the Appx low-atmo version better, but gaz is taken as canonical (verify against PDF p.131 if it matters).
- Minor single-digit divergences (TL or one physical) where gaz is used, Appx variant noted in-row: **Torpol** (gaz TL B / Appx 8), **Blue** (gaz B543487-B / Appx B443487-C), **Hilfer** (gaz B55077A-6 / Appx BA5077A-6), **Number One** (gaz C9D6778-9 / Appx C9C6778-7), **Marduk** (gaz C577436-5 / Appx C377436-3), **Thebus** (gaz -7 / Appx -6), **The World** (gaz -7 / Appx -6), **Fantasy** (gaz -3 / Appx -2), **Tech-World** (gaz A555154-E / Appx A455154-F), **Falcon** (gaz A558448-D / Appx A158448-D), **Sperle** (gaz BA8A78A-7 / Appx BA8A76A-7), **Umemii** (gaz -7 / Appx -6), **Ergo** (gaz X767500-3 / Appx X767500-0).
- Garbled-but-readable in the *current* source, flagged in-row, no clean cross-check available: **Scaladon** (Tobia) `AAD98DH-A` (law digit `H`), **Aiwewakh** (Hkea'as) `BA961G3-D` (gov digit `G`). Verify against the PoD2 PDF subsector pages if a precise digit is needed in play.
