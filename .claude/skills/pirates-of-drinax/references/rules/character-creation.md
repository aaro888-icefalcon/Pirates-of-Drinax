## Step-by-step overview (Core p.8)
1. Name the Traveller (do this first).
2. Roll 2D for each of 6 characteristics; note DMs. Start age 18.
3. Take background skills = EDU DM + 3 (each at level 0).
4. (Optional, term 1–3 only) Attempt University or Military Academy.
5. Enter a career: roll **qualification**. Fail → submit to the draft OR take Drifter this term.
6. Basic training (first career = all Service Skills at 0; later careers = one Service Skill at 0).
7. Each term (4 years): pick one skill table, roll 1D, gain that skill → roll **survival** → if survived, roll **events** → optionally roll **advancement** (and **commission** for military) → ageing check from term 4 on.
8. Repeat terms (same or new career) until you choose to muster out. Risks of ageing/injury rise; most PCs run 2–4 terms.
9. Muster out: Benefit rolls (cash + other benefits).
10. Apply the Connections rule, pick a group skills package, buy gear (≤Cr10000), finalize.

Do creation **as a group, term-by-term in parallel** so the Connections rule works. No combat/other situational DMs ever apply during creation — only the listed characteristic/skill DMs.

## Characteristics & DMs (Core p.9)
Roll 2D per characteristic, assign in any order (or in order STR→DEX→END→INT→EDU→SOC). Max 15 unaugmented.

| Char | Meaning |
|---|---|
| STR | physical strength, fitness, forcefulness |
| DEX | coordination, agility, reflexes |
| END | stamina, determination, damage capacity |
| INT | intelligence, quickness of mind |
| EDU | learning and experience |
| SOC | place in society |

**Characteristic DM table:**
| Score | DM |
|---|---|
| 0 | -3 |
| 1–2 | -2 |
| 3–5 | -1 |
| 6–8 | +0 |
| 9–11 | +1 |
| 12–14 | +2 |
| 15+ | +3 |

SOC ladder (noble titles, if desired): 11 Knight, 12 Baron, 13 Marquis, 14 Count, 15 Duke. High SOC ≠ automatically a noble — may be a celebrity, a relative of a noble, or a deposed/ousted noble (Core p.11).

## Background / homeworld skills (Core p.9)
Gain skills = **EDU DM + 3** (range 0–6), each at **level 0**, from this list. Pick to suggest a homeworld (agri world → Animals/Seafarer/Survival; belt → Vacc Suit/Electronics/Flyer).

`Admin, Animals, Art, Athletics, Carouse, Drive, Electronics, Flyer, Language, Mechanic, Medic, Profession, Science, Seafarer, Streetwise, Survival, Vacc Suit` (all at 0).

## Pre-career education (Core p.16) — first term only (can delay to term 2–3; none from term 4)
Pick University **or** Military Academy and roll for entry. Fail entry → must attempt a career immediately, else draft. One attempt per term. On entry, gain listed skills + roll Pre-career Events.

**University** — Entry **EDU 6+** (DM-1 term 2, DM-2 term 3; +1 if SOC 9+).
- Skills: choose one level-0 and one level-1 skill from: Admin, Advocate, Animals, Art, Astrogation, Electronics, Engineer, Language, Medic, Navigation, Profession, Science.
- Graduation **INT 7+** (11+ = honours). DM+1 if END 8+, DM+1 if SOC 8+.
- Grad benefits: +1 to each of the two chosen skills; EDU +1; **DM+1 (DM+2 honours) to qualify** for Agent/Army/Citizen(corporate)/Entertainer(journalist)/Marines/Navy/Scholar/Scouts; may take a **commission roll before term 1** of a first military career (honours = DM+2 on it; enter at officer rank 1 on success).

**Military Academy** — pick Army/Marines/Navy. Entry: Army **END 7+**, Marines **END 8+**, Navy **INT 8+** (DM-2 term 2, DM-4 term 3).
- Skills: all Service Skills of that service at level 0 (like basic training).
- Graduation **INT 7+** (honours separate). Grad benefits: raise any three Service Skills to 1; EDU +1; +1 SOC if honours; **automatic entry** into that military career (if first career after); commission roll before term 1 at **DM+2** (auto-pass if honours).
- Fail to graduate but roll **3+**: still auto-enter that service (no commission roll in term 1).

Pre-career Events table (2D, Core p.17) covers psionic recruitment, tragedy/non-graduation, pranks (Rival/Enemy), Carouse 1, D3 Allies, a free skill at 0, draft via war, SOC +1, etc. Summarize as flavor; apply the listed mechanical effect.

## The term cycle (each term = 4 years)
1. **Qualification** (Core p.18): roll the career's char check, **DM-1 per previous career** (some careers add age penalties). Fail → draft (once per life) or Drifter this term. Can't re-enter a career you just left next term (except via draft/Drifter). Noble auto-qualifies if SOC 10+.
2. **Basic training**: first career → gain *all* that career's Service Skills at level 0. Later careers → pick *one* Service Skill at 0. **Citizen & Drifter** use their chosen Assignment skill table for basic training, not Service Skills.
3. **Skills & training**: pick ONE table (Personal Development / Service / Assignment[one of 3] / Advanced Education / Officer), roll 1D, gain that entry. Advanced Education requires the listed EDU min (8 or 10). Officer table requires being commissioned.
   - No level listed = gain at 1 (or +1 if already trained). Level listed = set to that level if higher than current. E.g. "Gambler 0" only helps if you lack it; "Vacc Suit" = +1.
4. **Survival**: roll the assignment's survival check. **Natural 2 always fails.** Fail → roll on the career's **Mishap** table (1D) and leave the career immediately (unless the mishap says otherwise); lose this term's Benefit roll.
5. **Events** (if you survived): roll 2D on the career's Events table. Result 7 = Life Event (roll on Life Events). Result 2 = "Disaster" (roll Mishap but not ejected). Other entries grant skills, Contacts/Allies/Rivals/Enemies, Benefit-roll DMs, or advancement DMs.
6. **Advancement** (optional, once/term): roll the career's advancement check. Success → +1 rank, an extra skill-table roll this term, and any rank bonus. If the roll ≤ terms served in this career, you must leave after this term. **Natural 12 = must continue** (too valuable). Some events grant DMs or automatic advancement.
7. **Commission** (military only — Army/Marine/Navy, **SOC 8+**): optional; first term only unless SOC 9+ (then any term at DM-1 per term after the first). Success → become officer rank 1, switch to the Officer rank track. Gaining a commission means no advancement roll that same term (but you may still advance after a *failed* commission).
8. **Ageing**: from the end of term 4 (age 34) and every term after, roll on the Ageing table.

**Mishap mechanic (summary):** every career's Mishap table is 1D. Roll 1 ≈ "severely injured" (= Injury result 2, or roll Injury twice take lower). Roll 6 ≈ "roll on the Injury table". Rolls 2–5 are career-specific narrative ejections (gain Enemy/Rival, lose SOC, forced into Prisoner career on certain results, occasionally keep the Benefit roll). Apply the listed effect; most force you out of the career.

**Event mechanic (summary):** every career's Events table is 2D. 2 = Disaster (Mishap, no ejection); 7 = Life Event; 12-type result = automatic promotion. Mid-table entries give skills, associates, or +DMs. The connections rule (below) lets one Traveller's event involve another.

## Skill rules & limits (Core p.18)
- Cap of **level 4** during creation; excess lost.
- Total skill levels may never exceed **3 × (INT + EDU)**.
- Skill checks during creation add only that skill level (no characteristic/other DMs).

## Career skill tables — pirate-relevant careers
For each: Personal Development (PD), Service Skills (SS), the three Assignment columns, Advanced Education (AE, with EDU min). Roll 1D on the chosen column.

### Rogue (Core p.40) — corsair career
- **PD:** 1 Carouse, 2 DEX+1, 3 END+1, 4 Gambler, 5 Melee, 6 Gun Combat
- **SS:** 1 Deception, 2 Recon, 3 Athletics, 4 Gun Combat, 5 Stealth, 6 Streetwise
- **Thief:** Stealth / Electronics / Recon / Streetwise / Deception / Athletics
- **Enforcer:** Gun Combat / Melee / Streetwise / Persuade / Athletics / Drive
- **Pirate:** Pilot / Astrogation / Gunner / Engineer / Vacc Suit / Melee
- **AE (EDU 10):** Electronics / Deception / Navigation / Medic / Investigate / Broker / Advocate
- **Pirate ranks:** 0 Lackey; 1 Henchman (Pilot 1 or Gunner 1); 2 Corporal; 3 Sergeant (Gun Combat 1 or Melee 1); higher ranks (Lieutenant/Captain/Chief) per book.

### Scout (Core p.44) — ships, jump, recon
- **PD:** 1 STR+1, 2 DEX+1, 3 END+1, 4 INT+1, 5 EDU+1, 6 Jack-of-all-Trades
- **SS:** Pilot(small craft or spacecraft) / Survival / Mechanic / Astrogation / Vacc Suit / Gun Combat
- **Courier:** Electronics / Flyer / Pilot(spacecraft) / Engineer / Athletics / Astrogation
- **Surveyor:** Electronics / Persuade / Pilot / Navigation / Diplomat / Streetwise
- **Explorer:** Electronics / Pilot / Engineer / Science / Stealth / Recon
- **AE (EDU 8):** Medic / Language / Seafarer / Explosives / Science / Jack-of-all-Trades
- Ranks: 1 Scout (Vacc Suit 1), 3 Senior Scout (Pilot 1).

### Navy (Core p.36) — shipboard combat & ops
- **PD:** STR+1 / DEX+1 / END+1 / INT+1 / EDU+1 / SOC+1
- **SS:** Pilot / Vacc Suit / Athletics / Gunner / Mechanic / Gun Combat
- **Line/Crew:** Electronics / Mechanic / Gun Combat / Flyer / Melee / Vacc Suit
- **Engineer/Gunner:** Engineer / Mechanic / Electronics / Engineer / Gunner / Flyer
- **Flight:** Pilot / Flyer / Gunner / Pilot(small craft) / Astrogation / Electronics
- **AE (EDU 8):** Electronics / Astrogation / Engineer / Flyer / Medic / Admin
- **Officer:** Leadership / Electronics / Pilot / Melee(blade) / Admin / Tactics(naval)

### Marine (Core p.32) — boarding actions, ground assault
- **PD:** STR+1 / DEX+1 / Tactics / END+1 / Melee(unarmed) / Melee(blade)
- **SS:** Athletics / Vacc Suit / Heavy Weapons / Gun Combat / Stealth (+ Medic/Survival/Explosives/Engineer/Pilot/Navigation appear in AE)
- **Support:** Electronics / Mechanic / Drive or Flyer / Medic / Heavy Weapons / Gun Combat
- **Star Marine:** Vacc Suit / Athletics / Gunner / Melee(blade) / Electronics / Gun Combat
- **Ground Assault:** Vacc Suit / Heavy Weapons / Recon / Melee(blade) / Tactics(military) / Gun Combat
- **AE (EDU 8):** Medic / Survival / Explosives / Engineer / Pilot / Navigation
- **Officer:** Electronics / Tactics / Admin / Advocate / Diplomat / Leadership
- Enlisted rank 0 Marine grants Gun Combat(any) 1 or Melee(blade) 1.

### Merchant (Core p.34) — trade, free-trader crew
- **PD:** STR+1 / DEX+1 / END+1 / INT+1 / Language / Streetwise
- **SS:** Drive / Vacc Suit / Broker / Steward / Electronics / Persuade
- **Merchant Marine:** Pilot / Vacc Suit / Athletics / Mechanic / Engineer / Electronics
- **Free Trader:** Pilot(spacecraft) / Vacc Suit / Deception / Mechanic / Streetwise / Gunner
- **Broker:** Admin / Advocate / Broker / Streetwise / Deception / Persuade
- **AE (EDU 8):** Engineer / Astrogation / Electronics / Pilot / Admin / Advocate

### Agent (Core p.22) — intel, infiltration
- **PD:** Gun Combat / DEX+1 / END+1 / Melee / INT+1 / Athletics
- **SS:** Streetwise / Drive / Investigate / Flyer / Recon / Gun Combat
- **Law Enforcement:** Investigate / Recon / Streetwise / Stealth / Melee / Advocate
- **Intelligence:** Investigate / Recon / Electronics(comms) / Stealth / Persuade / Deception
- **Corporate:** Investigate / Electronics(computers) / Stealth / Carouse / Deception / Streetwise
- **AE (EDU 8):** Advocate / Language / Explosives / Medic / Vacc Suit / Electronics

(Army, Citizen, Drifter, Entertainer, Noble, Scholar skill tables follow the same column structure in Core p.24–43 if needed — Drifter has no Advanced Education and only PD+SS+3 assignments.)

## Mustering out (Core p.46)
Gain **1 Benefit roll per full term** in a career, plus rank bonuses. Each career has a **Cash** column and an **Other Benefits** column (roll 1D, choose which column per roll). **Cash column may be rolled at most 3 times total across all careers.** Gambler skill = DM+1 to Cash rolls.

**Bonus Benefit rolls by highest rank reached:** rank 1–2 = +1; 3–4 = +2; 5–6 = +3 and **DM+1 to all Benefit rolls from that career**.

**Other Benefits (shared definitions, Core p.47):**
- Characteristic increase: raise listed char (max 15; excess SOC becomes Ship Shares).
- Weapon (≤Cr3000/TL12), Gun (≤Cr3000), Blade (≤Cr1000) — duplicate = another or +1 relevant Combat skill.
- Armour (≤Cr10000/TL12; duplicate → ≤Cr25000).
- Cybernetic Implant (≤Cr75000/TL12).
- Scientific Equipment (≤Cr2000/TL12).
- TAS Membership (life; pays one High Passage every 2 months; duplicate = 2 Ship Shares).
- Ally / Contact.
- **Ship Shares:** each ≈ MCr1 toward a ship; cannot be cashed; pool with crew toward a mortgage. Unused share → Cr1000/yr pension instead.
- **Ships:** Scout Ship (Scout — still owned by Service, recallable for missions; reroll if gained twice); Free Trader / Far Trader (Merchant — 25% mortgage paid, roll 1D Quirks; four rolls = paid off); Lab Ship, Yacht/Safari, Ship's Boat (≤MCr10) etc. **Only ONE Traveller may start the campaign with a ship**; others who rolled a ship get +Cr25000/yr pension per such roll. *(In Pirates of Drinax the crew is granted the Harrier-class instead — these personal ships are usually backstory or sold off.)*

**Spending:** after mustering out, buy personal gear up to **Cr10000**; leftover cash can be kept, converted to trade goods, or saved.

**Pension (Core p.48):** leaving a career (NOT Scout/Rogue/Prisoner/Drifter) after **≥5 terms** = retired, paid at start of each year at any Class A/B starport: 5 terms Cr10000, 6 Cr12000, 7 Cr14000, 8 Cr16000, +Cr2000/term beyond 8.

## Ageing (Core p.48)
From end of term 4 (age 34) and every term after, roll 2D on the Ageing table with **DM = − total terms**. Player chooses which characteristics take the hit.

| 2D (after DM) | Effect |
|---|---|
| -6 | -2 to three physical chars, -1 to one mental |
| -5 | -2 to three physical chars |
| -4 | -2 to two physical, -1 to one physical |
| -3 | -2 to one physical, -1 to two physical |
| -2 | -1 to three physical |
| -1 | -1 to two physical |
| 0 | -1 to one physical |
| 1+ | no effect |

**Ageing crisis:** any characteristic reduced to 0 → die unless you pay 1D×Cr10000 (restores affected chars to 1); thereafter auto-fail all qualification rolls. **Anagathics** (start a term by rolling SOC 10+; roll exactly 2 → Prisoner career): stop ageing but require **two** survival checks per term, cost 1D×Cr25000/term (paid from cash benefits; unpaid → debt).

## Injury (Core p.49)
Combat/accident injuries roll 1D: 1 Nearly killed (one physical -1D, two others -2); 2 Severely injured (one physical -1D); 3 Missing eye/limb (STR or DEX -2); 4 Scarred (any physical -2); 5 Injured (any physical -1); 6 Lightly injured (no permanent effect). Restore a lost characteristic point = Cr5000/point; service may pay part via the Medical Bills table (DM = rank).

## Connections, contacts, skills packages (Core p.19, 46, 50)
**Connections rule:** during creation, any event one Traveller rolls can involve another (only one must have rolled it). Each connection = both Travellers gain one free skill of choice (max **two** free skills per Traveller, each connection with a different Traveller; cannot raise a skill above 3 this way; no Jack-of-all-Trades). Builds shared history so the crew already knows each other.

**Associates:** Contact (friendly, useful), Ally (actively helps), Rival (opposes within the law), Enemy (wants you dead/ruined). Note each as it's gained.

**Life Events (2D, Core p.46):** 2 sickness/injury; 3 birth/death; 4 ended relationship (Rival/Enemy); 5 improved relationship (Ally); 6 new relationship (Ally); 7 new Contact; 8 betrayal (Contact/Ally→Rival/Enemy); 9 travel (DM+2 next qualification); 10 good fortune (DM+2 a Benefit roll); 11 crime (lose a Benefit roll or Prisoner next term); 12 unusual event (1D: psionics / aliens / alien artefact / amnesia / govt contact / Ancient tech).

**Group skills package (after everyone finishes):** all Travellers collectively pick ONE package; take items in turn until all are assigned (fills gaps). Options:
- **Traveller (all-round):** Deception 1, Electronics 1, Gun Combat 1, Gunner 1, Medic 1, Persuade 1, Pilot 1, Stealth 1
- **Trader:** Advocate 1, Astrogation 1, Broker 1, Diplomat 1, Electronics 1, Medic 1, Pilot 1, Streetwise 1
- **Explorer:** Astrogation 1, Electronics 1, Gun Combat 1, Medic 1, Pilot 1, Recon 1, Stealth 1, Survival 1
- **Investigator:** Admin 1, Advocate 1, Deception 1, Electronics 1, Gun Combat 1, Investigate 1, Persuade 1, Stealth 1, Streetwise 1
- **Mercenary:** Electronics 1, Medic 1, Leadership 1, Heavy Weapons 1, Gun Combat 1×2, Stealth 1, Recon 1
- **Starship:** Astrogation 1, Electronics 1, Engineer 1, Gunner 1, Mechanic 1, Medic 1, Pilot 1, Tactics(naval) 1
- **Criminal:** Broker 1, Deception 1, Electronics 1, Medic 1, Persuade 1, Pilot 1, Stealth 1, Streetwise 1
- **Diplomat:** Admin 1, Advocate 1, Deception 1, Diplomat 1, Electronics 1, Persuade 1, Stealth 1, Streetwise 1

## Final assembly of a PC
1. Lock characteristics (after all ageing/injury/benefit increases); recompute DMs.
2. Total skills (apply 4-cap and the 3×(INT+EDU) cap).
3. Total cash (≤3 Cash rolls used); resolve any medical/anagathic debt; buy gear ≤Cr10000.
4. Record benefits (gear, augments, TAS, Ship Shares, ship). Resolve who keeps the one allowed ship (others get pension).
5. List Contacts/Allies/Rivals/Enemies and connection-derived skills.
6. Apply the group skills package.
7. Note age, terms served, pension (if ≥5-term retiree), title (if SOC ≥11 and desired).

## Best careers for a pirate crew (Drinax)
- **Rogue (Pirate):** the archetype — Pilot, Astrogation, Gunner, Engineer, Vacc Suit, Melee; gives Ship Shares and weapons. Core pirate skillset.
- **Navy** & **Marine:** Navy = pilots, gunners, engineers, naval tactics; Marine (Star Marine) = boarding actions, Vacc Suit, Gun Combat, Tactics — exactly what raiding/prize-taking needs.
- **Scout:** Pilot, Astrogation, Navigation, Survival, Jack-of-all-Trades, sensor/recon work, and (rarely) a hull — superb for navigating the Trojan Reach frontier.
- **Merchant (Free Trader/Broker):** Pilot, Broker, Streetwise, Steward — sells loot, runs the legitimate front, handles passengers/fences.
- **Agent (Intelligence/Corporate):** infiltration, Deception, Streetwise, Investigate — scouting targets and ports.
- **Drifter (Scavenger):** belters/salvagers with Pilot(small craft), Astrogation, Mechanic, Vacc Suit; always available (auto-qualify) and feeds Ship Shares — a fine fallback to round out a crew.
Engineering/medical gaps are best filled by Navy/Scout (Engineer, Mechanic) and Scholar (Physician) or the Starship/Traveller skills package.
