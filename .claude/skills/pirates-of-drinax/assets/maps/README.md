## Note on the Trojan Reach hex map
A clean full-sector **hex map** is not included here: in the source it is a label-dense spread that did not render cleanly as a standalone image. **All world data is preserved in text** in `references/setting/worlds/00_world-index.md` (every world's hex, UWP, bases, trade codes, zone, allegiance). For the visual hex map, see the original *Pirates of Drinax* Book 2 / the bundled `sources/Pirates_of_Drinax_2_ebook.md` figure references. More full-page plates per book remain in the user's converted `Traveller_Markdown/<book>_images/` folders.
