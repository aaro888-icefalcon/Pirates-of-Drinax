# Seed Scout (portable prompt)
The Scout is OPTIONAL. By default the **main AI** refreshes the seed deck inline at bookkeeping;
offload to a subagent only when convenient (e.g., long canon). Use the `mythic-scout` agent in
Claude Code, or spawn a Task with the prompt below in Cowork.

> Refresh `<campaign>/seeds.md` for this solo Mythic campaign. Sources & size are in the bridge `seeds.md`
> (30-40 seeds). Pull from: setting-canon near the PC; live world state (active clocks, revealed regions,
> faction standings); and random rolls on the bridge generators (`dice.py table <path>` — show rolls).
> Also harvest any new Threads/Characters/Features the just-ended scene touched (canon-cited, PC-earned only).
> Tag each seed with the threads/characters/elements it touches + its source. Keep un-used relevant seeds,
> drop stale ones. This enriches the pool only — the dice still decide what's used.
