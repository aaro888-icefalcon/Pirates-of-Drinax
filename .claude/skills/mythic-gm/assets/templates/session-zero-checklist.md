# Session Zero checklist

- [ ] Confirm the player wants honest, hardcore play (real consequences, no rescues).
- [ ] **RPG ruleset:** built-in knowledge / uploaded rulebook / rules-light → build `system-profile.md` if a system is named.
- [ ] **Setting / lore:** from user or second skill → `setting-canon.md`, or generate one.
- [ ] **Genre & stakes vocabulary:** pick a `references/genres/` pack.
- [ ] **Adventure Source mode:** Pure Mythic / Adventure Crafter / Prepared Adventure.
- [ ] **Resolution:** Fate Chart (default) or Fate Check; Chaos flavor.
- [ ] **Overlays:** Keyed Scenes? Thread Progress Track? Peril Points (off by default)?
- [ ] **Chaos Factor = 5**, empty Threads/Characters Lists (+ Adventure Features if prepared).
- [ ] **Create the PC** → `character-sheet.md`; seed 1–2 Threads from backstory.
- [ ] **First Scene** (not tested) per mode; seed the Lists; "What do you do?" → STOP.
- [ ] Write everything into `campaign-state.md` (copy of the state template).
